# Contributing to TaoApex

Thank you for your interest in contributing to [TaoApex](https://taoapex.com/)!

## How to Contribute

Since TaoApex is proprietary software, we don't accept code contributions. However, there are many other ways you can help:

### Report Bugs
Found a bug? Please create an issue on [GitHub Issues](https://github.com/taoapex/taoapex/issues) with:
- A clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable
- Browser/device information

### Request Features
Have an idea for a new feature? Share it in [GitHub Discussions](https://github.com/taoapex/taoapex/discussions)!

### Improve Documentation
- Fix typos or unclear explanations
- Add examples or use cases
- Translate documentation

### Spread the Word
- Star this repository
- Share [TaoApex](https://taoapex.com/) with friends and colleagues
- Write about your experience using our products:
  - [Imagine](https://taoapex.com/en/products/imagine/) - AI Headshot Generator
  - [Talk](https://taoapex.com/en/products/talk/) - AI Companion
  - [PromptHub](https://taoapex.com/en/products/prompt/) - Prompt Manager

## Documentation Contributions

If you'd like to improve our documentation:

1. Fork this repository
2. Make your changes
3. Submit a pull request

We welcome improvements to:
- Typo fixes
- Clarifications
- Additional examples
- Translations

## Code of Conduct

Please be respectful and constructive in all interactions. We're building a positive community around TaoApex.
